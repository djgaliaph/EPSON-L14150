Comprobar el SSID y la contraseña del router inalámbrico

Para encontrar el SSID y la contraseña, compruebe la parte posterior o inferior del router inalámbrico al que desea conectarse, o la documentación que se proporciona con él, como una tarjeta de información, una etiqueta o un manual.
Si se muestra un SSID con un nombre que es similar al que desea conectarse, por ejemplo, si el nombre del SSID difiere por una letra, como a o g, o un número como 2.4 o 5, selecciónelo.
Si no se muestra el SSID al que desea conectarse, siga las instrucciones del panel de control de la impresora para actualizar la lista de SSID buscando nuevamente. Si aún no se muestra, introduzca el SSID manualmente.
