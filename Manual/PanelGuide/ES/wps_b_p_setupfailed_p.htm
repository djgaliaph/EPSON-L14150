Error de configuración

Compruebe los siguientes elementos y, a continuación, vuelva a realizar la configuración de red.
Compruebe que el router inalámbrico está encendido. Si está encendido, apague y vuelva a encender la alimentación del router inalámbrico. Tenga en cuenta a otros usuarios que puedan estar utilizando la red.
Deshabilite el separador de privacidad en el router inalámbrico.
Coloque la impresora cerca del router inalámbrico.
Compruebe que el router inalámbrico admite la funcionalidad WPS. Si no admite la funcionalidad WPS, no podrá establecer una red mediante la funcionalidad Configuración de pulsador (WPS).
Si la función de filtrado de direcciones MAC está habilitada, registre la dirección MAC de la impresora para evitar que se filtre. Consulte la documentación proporcionada con el router inalámbrico para el registro.
Compruebe que el modo de comunicación Wi-Fi (IEEE802.11*) establecido para el router inalámbrico coincide con el que admite la impresora. (*La información de la pantalla varía conforme a su ubicación.)
Cuando conecte un dispositivo al puerto USB 3.0 de un equipo Mac, se pueden producir interferencias de radiofrecuencia. Pruebe lo siguiente si no puede conectarse a la red LAN inalámbrica (Wi-Fi) o si las operaciones se vuelven inestables.
Coloque el dispositivo que esté conectado al puerto USB 3.0 alejado del equipo.
Si la impresora admite el intervalo de frecuencia de 5 GHz, conéctese al SSID para el intervalo de 5 GHz.
