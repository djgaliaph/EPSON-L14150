Escribir la contraseña y conectarse

1 . Conéctese al SSID.

Para Windows 10
Desactive Conectar automáticamente y, a continuación, haga clic en Conectar.
Para Windows 8.1, Windows 8, Windows 7 y Windows Vista
Haga clic en Conectar.
Para Windows XP
Haga clic con el botón derecho en SSID y, a continuación, haga clic en Conectar.
Para MacOS
No se requiere ninguna acción.

2 . Conecte la impresora y el equipo introduciendo la contraseña que se muestra en el panel de control de la impresora conectada al equipo.

Para Windows XP, escriba la clave de red y la clave de confirmación de red.

3 . Para usuarios de Windows 8.1, Windows 8, Windows 7 y Windows Vista, puede configurar el equipo para conectarse a la impresora automáticamente cada vez que encienda su equipo.

No tiene que establecer esta opción si solamente quiere conectar la impresora cuando la utilice.
