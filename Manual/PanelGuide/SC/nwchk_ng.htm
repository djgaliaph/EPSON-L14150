网络连接报告上的消息和解决方案

打印机与无线路由器之间的连接有问题。检查网络连接报告中的消息和错误代码，然后按照解决方案执行。

使打印机靠近无线路由器并移除二者之间的障碍物。
如果手动输入 SSID，请检查是否正确。检查网络连接报告 Network Status 部分中的 SSID。
如果一个无线路由器有多个 SSID，请选择显示的 SSID。如果 SSID 使用不合规频率，打印机不会显示。
如果您使用 Push Button Setup 建立网络连接，确保无线路由器支持 WPS。如果无线路由器不支持 WPS，则您不能使用 Push Button Setup。
确保您的 SSID 只使用 ASCII 字符（字母数字字符和符号）。打印机无法显示包含非 ASCII 字符的 SSID。
在连接无线路由器前，确保 SSID 和密码已知。如果使用默认设置的无线路由器，SSID 和密码可在无线路由器的标签上找到。如果 SSID 和密码未知，请联系设置无线路由器的人员，或参阅无线路由器随附的文档。
要连接从共享网络的智能设备中生成的 SSID，请检查智能设备随附文档上的 SSID 和密码。
如果 Wi-Fi 连接突然断开，请检查以下条件。如果符合任何一个条件，请从以下网站下载和运行此软件以重置网络设置。
http://epson.sn > 设置

E-5
确保无线路由器的安全类型设为以下一个。如果不是，请更改无线路由器上的安全类型，然后重置打印机的网络设置。
WEP-64 bit (40 bit)
WEP-128 bit (104 bit)
WPA PSK (TKIP/AES)*
WPA2 PSK (TKIP/AES)*
WPA (TKIP/AES)
WPA2 (TKIP/AES)
WPA3-SAE (AES)
* WPA PSK 也称为 WPA Personal。WPA2 PSK 也称为 WPA2 Personal。
E-6
检查 MAC 地址过滤是否禁用。如果启用，请注册打印机的 MAC 地址以使其不会被过滤。请参阅无线路由器随附的文档了解详情。您可以在网络连接报告上的 Network Status 部分中检查打印机的 MAC 地址。
如果无线路由器使用 WEP 安全的共享验证，请确保验证密钥和索引正确。
如果无线路由器上可连设备数少于要连接的网络设备数，请设置无线路由器增加可连设备数。请参阅无线路由器随附的文档进行设置。
E-8
当打印机的“获得 IP 地址”设置设为自动时在无线路由器上启用 DHCP。
如果打印机的“获得 IP 地址”设置设为手动，则手动设置 IP 地址会因为超出范围（例如：0.0.0.0）而无效。从打印机控制面板或使用 Web Config 设置有效 IP 地址。
E-9
检查以下各项。
设备已打开。
您可以从要连接到打印机的设备，访问 Internet 和同一网络上的其它计算机或网络设备。
如果在确认上面各项后仍没有连接打印机和网络设备，请关闭无线路由器。等待约 10 秒后再打开。然后，从以下网站下载和运行此安装程序以重置网络设置。
http://epson.sn > 设置
E-10
检查以下各项。
网络上的其它设备已打开。
如果已将打印机的“获得 IP 地址”设为手动，则网络地址（IP 地址、子网掩码和默认网关）正确。
如不正确，请重置网络地址。您可以在网络连接报告上的 Network Status 部分中检查 IP 地址、子网掩码和默认网关。
如果 DHCP 已启用，将打印机的“获得 IP 地址”设置更改为自动。要手动设置 IP 地址，请从网络连接报告的 Network Status 部分中检查打印机的 IP 地址，然后在网络设置屏幕上选择手动。将子网掩码设为 [255.255.255.0]。
如果仍没有连接打印机和网络设备，请关闭无线路由器。等待约 10 秒后再打开。
E-11
检查以下各项。
将打印机 TCP/IP 设置设为手动时，默认网关地址是正确的。
设为默认网关的设备已打开。
设置正确的默认网关地址。您可以在网络连接报告上的 Network Status 部分中检查默认网关地址。
E-12
检查以下各项。
网络上的其它设备已打开。
如果手动输入，则网络地址（IP 地址、子网掩码和默认网关）正确。
其它设备的网络地址（子网掩码和默认网关）相同。
IP 地址不与其它设备冲突。
如果在确认上面各项后仍没有连接打印机和网络设备，请尝试以下操作。
关闭无线路由器。等待约 10 秒后再打开。
使用安装程序再次设置网络。您可以从以下网站运行。
http://epson.sn > 设置
您可以在使用 WEP 安全类型的无线路由器上注册几个密码。如果注册几个密码，请检查在打印机上是否设置第一个注册的密码。
E-13
检查以下各项。
无线路由器、集线器和路由器等网络设备已打开。
网络设备的 TCP/IP 设置未手动设定。(如果打印机 TCP/IP 设置自动设定而其它网络设备的 TCP/IP 设置手动执行，则打印机网络可能不同于其它设备的网络。)
如果检查以上各项后仍不管用，请尝试以下操作。
关闭无线路由器。等待约 10 秒后再打开。
使用安装程序，对与打印机在相同网络上的计算机进行网络设置。您可以从以下网站运行。
http://epson.sn > 设置
您可以在使用 WEP 安全类型的无线路由器上注册几个密码。如果注册几个密码，请检查在打印机上是否设置第一个注册的密码。

The same SSID may be set on several wireless routers. Check the settings on the wireless router, and change the SSID.
在几个无线路由器上可能设置了同一 SSID。检查无线路由器上的设置并更改 SSID。
The Wi-Fi environment needs to be improved. Turn the wireless router off and then turn it on. If the connection does not improve, see the documentation for the wireless router.
使打印机靠近无线路由器并移除它们之间的障碍后，关闭无线路由器。等待约 10 秒后再打开。如果仍不连接，请参阅无线路由器随附的文档。
*No more devices can be connected. Disconnect one of the connected devices if you want to add another one.

您可以通过检查网络状态页确认可同时连接的无线设备数和已连接的设备数。
