输入密码并连接

1 . 连接到 SSID。

对于 Windows 10
不选中自动连接，然后单击连接。
对于 Windows 8.1/Windows 8/Windows 7/Windows Vista
单击连接。
对于 Windows XP
右击 SSID，然后单击连接。
对于 MacOS
不需要操作。

2 . 通过将打印机控制面板上显示的密码输入到计算机，连接打印机和计算机。

对于 Windows XP，输入网络密钥和确认网络密钥。

3 . 对于 Windows 8.1/Windows 8/Windows 7/Windows Vista 用户，您可以设置计算机在每次打开时自动连接到打印机。

如果只想在使用打印机时连接，则不必进行该设置。
