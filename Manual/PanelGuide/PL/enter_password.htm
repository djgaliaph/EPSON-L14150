Wprowadzanie hasła i nawiązywanie połączenia

1 . Połącz z SSID.

System Windows 10
Wyczyść komunikat Połącz automatycznie, a następnie kliknij Połącz.
Systemy Windows 8.1/Windows 8/Windows 7/Windows Vista
Kliknij przycisk Połącz.
System Windows XP
Kliknij prawym przyciskiem SSID, a następnie kliknij polecenie Połącz.
System MacOS
Nie jest wymagane żadne działanie.

2 . Połącz drukarkę i komputer, wprowadzając do komputera hasło wyświetlone w panelu sterowania drukarki.

W systemie Windows XP wprowadź Klucz sieciowy i Potwierdzenie klucza sieciowego.

3 . Użytkownicy systemów Windows 8.1/Windows 8/Windows 7/Windows Vista mogą tak skonfigurować komputer, aby przy każdym włączeniu komputera następowało automatyczne łączenie z drukarką.

Nie jest konieczne włączanie tej opcji, jeśli połączenie z drukarką ma być nawiązywane tylko wtedy, gdy drukarka jest potrzebna.
