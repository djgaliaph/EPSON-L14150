Salasanan syöttäminen ja yhdistäminen

1 . Yhdistä SSID:hen.

Windows 10
Tyhjennä Yhdistä automaattisesti ja napsauta Yhdistä.
Windows 8.1 / Windows 8 / Windows 7 / Windows Vista
Napsauta Yhdistä.
Windows XP
Napsauta hiiren oikealla painikkeella SSID-valintaa ja napsauta sitten Yhdistä.
MacOS
Mikään toiminta ei ole tarpeen.

2 . Yhdistä tietokone ja tulostin syöttämällä tulostimen ohjauspaneelissa näkyvä salasana tietokoneeseen.

Windows XP: syötä Verkkoavain ja Vahvista verkkoavain.

3 . Windows 8.1 /Windows 8 /Windows 7 / Windows Vista -käyttäjät: voit asettaa tietokoneen muodostamaan automaattisesti yhteyden tulostimeen joka kerta, kun käynnistät tietokoneen.

Tätä ei tarvitse asettaa, jos haluat muodostaa yhteyden tulostimeen vain, kun käytät sitä.
