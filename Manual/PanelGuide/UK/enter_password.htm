Введення паролю та підключення

1 . Підключіть пристрій до SSID.

Для ОС Windows 10
Очистіть поле Підключатися автоматично, а потім натисніть кнопку Підключити.
Для ОС Windows 8.1/Windows 8/Windows 7/Windows Vista
Натисніть кнопку Підключити.
Для ОС Windows XP
Клацніть правою клавішею миші по SSID, а потім натисніть кнопку Підключити.
Для ОС MacOS
Не потрібно жодних дій.

2 . Підключіть принтер і комп'ютер, ввівши пароль, що відображається на панелі керування принтером, на комп'ютері.

Для ОС Windows XР: введіть ключ мережі та натисніть кнопку Підтвердити код мережі.

3 . Для користувачів ОС Windows 8.1/Windows 8/Windows 7/Windows Vista: ви можете налаштувати комп'ютер на підключення до принтера автоматично під час кожного ввімкнення комп'ютера.

Не встановлюйте цю функцію, якщо ви хочете підключати принтер, лише коли його використовуєте.
