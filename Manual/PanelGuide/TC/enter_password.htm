輸入密碼並連線

1 . 連線到 SSID。

適用於 Windows 10
清除[自動連線]，然後按一下 [連線]。
適用於 Windows 8.1/Windows 8/Windows 7/Windows Vista
按一下[連線]。
適用於 Windows XP
在 SSID 上按滑鼠右鍵，然後按一下[連線]。
適用於 MacOS
無需採取動作。

2 . 將印表機控制面板上顯示的密碼輸入電腦，使印表機與電腦連線。

若為 Windows XP，請輸入[網路金鑰]及[確認網路金鑰]。

3 . 若為 Windows 8.1/Windows 8/Windows 7/Windows Vista 使用者，您可將電腦設為每次開機時自動連線到印表機。

若您僅想要在使用印表機時連線到印表機，您無須設定此選項。
