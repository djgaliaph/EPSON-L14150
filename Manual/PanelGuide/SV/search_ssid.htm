Visa ett SSID i datorn

1 . Öppna inställningsskärmen för SSID du vill ansluta till.

För Windows 10
Klicka på på aktivitetsfältet.
Tillgängliga SSID visas på skärmen.
För Windows 8.1/Windows 8/Windows 7
Klicka på på aktivitetsfältet.
Tillgängliga SSID visas på skärmen.
För Windows Vista
Högerklicka på det trådlösa nätverkets ikon på aktivitetsfältet och klicka sedan på Visa tillgängliga trådlösa nätverk.
Tillgängliga SSID visas på skärmen Anslut till trådlöst nätverk.
För Windows XP
Klicka på Start > Kontrollpanel > Nätverksanslutningar > Skapa en ny anslutning.
Tillgängliga SSID visas på skärmen Trådlös nätverksanslutning.
För MacOS
Klicka på menyn .
Tillgängliga SSID visas på skärmen.

2 . Välj samma SSID som visas på skrivarens kontrollpanel.
