Ange lösenord och ansluta

1 . Anslut till SSID.

För Windows 10
Rensa Anslut automatiskt och klicka sedan på Anslut.
För Windows 8.1/Windows 8/Windows 7/Windows Vista
Klicka på Anslut.
För Windows XP
Högerklicka på SSID och klicka sedan på Anslut.
För MacOS
Ingen åtgärd krävs.

2 . Anslut skrivaren och datorn genom att skriva in lösenordet på skrivarens kontrollpanel i datorn.

För Windows XP, ange Nätverksnyckel och Bekräfta nätverksnyckel.

3 . Användare av Windows 8.1/Windows 8/Windows 7/Windows Vista kan ställa in datorn så att den ansluts automatiskt till skrivaren varje gång du startar datorn.

Du behöver inte ställa in detta om du endast vill ansluta till skrivaren när du använder den.
