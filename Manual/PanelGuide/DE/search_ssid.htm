So zeigen Sie eine SSID am Computer an

1 . Öffnen Sie den Auswahlbildschirm der SSID, zu der Sie die Verbindung durchführen möchten.

Für Windows 10
Klicken Sie auf in der Taskleiste.
Die verfügbaren SSIDs werden auf dem Bildschirm angezeigt.
Für Windows 8.1/Windows 8/Windows 7
Klicken Sie auf in der Taskleiste.
Die verfügbaren SSIDs werden auf dem Bildschirm angezeigt.
Für Windows Vista
Rechtsklicken Sie auf das Drahtlos-Symbol in der Taskleiste und klicken Sie dann auf Verfügbare Drahtlosnetzwerke anzeigen.
Die verfügbaren SSIDs werden auf dem Bildschirm Verbinden mit dem Drahtlosnetzwerk angezeigt.
Für Windows XP
Klicken Sie auf Start > Bedienfeld > Netzwerkverbindungen > Neue Verbindung herstellen.
Die verfügbaren SSIDs werden auf dem Bildschirm Drahtlosnetzwerk-Verbindung angezeigt.
Für MacOS
Klicken Sie auf das Menü .
Die verfügbaren SSIDs werden auf dem Bildschirm angezeigt.

2 . Wählen Sie dieselbe SSID wie die, die auf dem Bedienfeld des Druckers angezeigt wird.
