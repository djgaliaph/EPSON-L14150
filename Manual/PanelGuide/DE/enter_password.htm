Eingeben des Passworts und Verbinden

1 . Stellen Sie die Verbindung mit SSID her.

Für Windows 10
Wählen Sie Automatisch verbinden ab, klicken Sie dann auf Verbinden.
Für Windows 8.1/Windows 8/Windows 7/Windows Vista
Klicken Sie auf Verbinden.
Für Windows XP
Rechtsklicken Sie auf die SSID und klicken Sie dann auf Verbinden.
Für MacOS
Keine Aktion erforderlich.

2 . Verbinden Sie den Drucker und den Computer, indem Sie das Passwort, das auf dem Bedienfeld des Druckers angezeigt wird, im Computer eingeben.

Für Windows XP geben Sie den Netzwerkschlüssel und Netzwerkschlüssel bestätigen ein.

3 . Die Benutzer von Windows 8.1/Windows 8/Windows 7/Windows Vista können ihren Computer so einstellen, dass die Verbindung zum Drucker jedes Mal, wenn sie den Computer verwenden, automatisch hergestellt wird.

Sie müssen diese Einstellung nicht vornehmen, wenn Sie die Verbindung zum Drucker nur herstellen wollen, wenn Sie diesen verwenden.
