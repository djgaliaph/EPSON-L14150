Fehlersuche

Für Computer
Falls Sie durch Sicherheitssoftware benachrichtigt werden, dass die Kommunikation mit EpsonNet Setup blockiert wird, müssen Sie die Kommunikation erlauben.
Ändern Sie die Einstellungen der Sicherheitssoftware, um die Kommunikation mit EpsonNet Setup über die Firewall zu erlauben.
Für Wi-Fi
Schalten Sie den Wireless-Router aus und wieder ein. Beachten Sie bitte, dass andere Nutzer das Netzwerk verwenden könnten.
Klicken Sie auf Details und drucken Sie dann einen Netzwerkverbindungsbericht. Überprüfen Sie die Probleme und Abhilfemaßnahmen.
Führen Sie je nach Diagnoseergebnis die entsprechenden Abhilfemaßnahmen unter dem jeweiligen Link durch.

For [FEHLER]
Wenn Sie ein Gerät mit dem USB-3.0-Anschluss an einem Mac verbinden, können Funkfrequenzstörungen auftreten. Versuchen Sie Folgendes, falls Sie keine Verbindung zum WLAN (Wi-Fi) herstellen können oder Vorgänge instabil werden.
Stellen Sie das mit dem USB-3.0-Anschluss verbundene Gerät in größerer Entfernung zum Computer auf.
Falls der Drucker den 5-GHz-Frequenzbereich unterstützt, stellen Sie für den 5-GHz-Bereich eine Verbindung zur SSID her.
Für drahtgebundenes LAN
Schalten Sie den Hub aus und wieder ein. Beachten Sie bitte, dass andere Nutzer das Netzwerk verwenden könnten. Überprüfen Sie die Verbindung erneut ca. eine Minute nach Aus- und Wiedereinschalten des Hubs.
Vergewissern Sie sich, dass das Ethernet-Kabel sicher angeschlossen ist. Wenn die Verbindungsstatusanzeige erloschen ist, überprüfen Sie den Anzeigenstatus, indem Sie einen anderen Port oder ein anderes Ethernet-Kabel verwenden. Falls das Problem nicht behoben wird, ersetzen Sie den Hub.
