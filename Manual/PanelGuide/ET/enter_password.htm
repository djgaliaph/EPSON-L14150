Parooli sisestamine ja ühenduse loomine

1 . Ühenduse loomine SSID-iga.

Windows 10
Tühjendage ruut Ühenda automaatselt ja seejärel klõpsake käsku Ühenda.
Windows 8.1 / Windows 8 / Windows 7 / Windows Vista
Klõpsake käsku Loo ühendus.
Windows XP
Paremklõpsake suvandit SSID ja seejärel klõpsake käsku Loo ühendus.
MacOS
Toimingute tegemine pole nõutav.

2 . Ühendage printer ja arvuti, sisestades arvutisse printeri juhtpaneelil kuvatud parool.

Windows XP korral sisestage valikud Võrgu võti ja Kinnita võrgu võti.

3 . Windows 8.1 / Windows 8 / Windows 7 / Windows Vista korral saavad kasutajad seadistada arvuti printeriga automaatselt ühendatavaks iga kord, kui arvuti sisse lülitatakse.

Ärge seda määrake, kui soovite luua printeriga ühenduse ainult printeri kasutamisel.
