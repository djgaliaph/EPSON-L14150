암호 입력 및 연결

1 . SSID에 연결합니다.

Windows 10의 경우
자동으로 연결을 선택 취소한 다음 연결을 클릭합니다.
Windows 8.1/Windows 8/Windows 7/Windows Vista의 경우
연결을 클릭합니다.
Windows XP의 경우
SSID를 오른쪽 클릭한 다음 연결을 클릭합니다.
MacOS의 경우
필요한 작업이 없습니다.

2 . 프린터 제어판에 표시된 암호를 컴퓨터에 입력하여 프린터와 컴퓨터를 연결합니다.

Windows XP의 경우 네트워크 키 및 네트워크 키 확인을 입력합니다.

3 . Windows 8.1/Windows 8/Windows 7/Windows Vista 사용자의 경우 컴퓨터를 켤 때마다 컴퓨터가 프린터에 자동으로 연결되도록 설정할 수 있습니다.

사용할 때만 프린터에 연결되도록 하려면 이 항목을 설정하지 마십시오.
