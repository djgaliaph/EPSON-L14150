네트워크 연결 보고서의 메시지 및 해결 방법

프린터와 무선 라우터 간의 연결에 문제가 있습니다. 네트워크 연결 보고서의 메시지 및 오류 코드를 확인한 다음 해결 방법을 따르십시오.
a: 오류 코드
코드
해결 방법
E-1
이더넷 케이블이 프린터, 허브 또는 기타 네트워크 장치에 단단히 연결되었는지 확인합니다.
허브 또는 기타 네트워크 장치가 켜져 있는지 확인합니다.
프린터에 Wi-Fi로 연결하려면, 프린터의 Wi-Fi 설정을 다시 설정합니다. 비활성화되어 있기 때문입니다.

E-7
무선 라우터가 켜져 있는지 확인합니다.
컴퓨터 또는 장치가 무선 라우터에 올바로 연결되었는지 확인합니다.
무선 라우터를 끕니다. 10초 정도 기다렸다가 다시 켭니다.
프린터를 무선 라우터에 더 가까이 놓고 프린터와 무선 라우터 사이의 장애물을 모두 제거합니다.
SSID를 수동으로 입력한 경우 올바른지 확인합니다. 네트워크 연결 보고서의 Network Status 부분에서 SSID를 확인합니다.
무선 라우터의 SSID가 여러 개일 경우, 표시된 SSID를 선택합니다. SSID가 비표준 주파수를 사용하고 있을 경우, 프린터에 표시되지 않습니다.
푸시 버튼 설정을 사용하여 네트워크 연결을 하고 있을 경우, 무선 라우터가 WPS를 지원하는지 확인합니다. 무선 라우터가 WPS를 지원하지 않을 경우 푸시 버튼 설정을 사용할 수 없습니다.
SSID가 ASCII 문자(영숫자 문자와 기호)만 사용하는지 확인합니다. 프린터는 ASCII 문자 이외의 문자가 포함된 SSID를 표시할 수 없습니다.
무선 라우터에 연결하기 전에 SSID와 암호를 확인합니다. 기본 설정으로 설정된 무선 라우터를 사용 중인 경우 SSID 및 암호가 무선 라우터의 라벨에 표시되어 있습니다. SSID 및 암호를 모르는 경우 무선 라우터 설치 담당자에게 문의하거나 무선 라우터와 함께 제공된 설명서를 참조하십시오.
테더링 스마트 장치에서 생성한 SSID에 연결하려면 스마트 장치와 함께 제공된 설명서에서 SSID 및 암호를 확인합니다.
Wi-Fi 연결이 갑자기 끊긴 경우 아래의 조건을 확인하십시오. 이러한 조건들 중 어느 하나라도 적용될 수 있을 경우, 다음 웹사이트에서 소프트웨어를 다운로드하고 실행하여 네트워크 설정을 재설정하십시오.
http://epson.sn > 설정
다른 스마트 장치가 푸시 버튼 설정을 사용하여 네트워크에 추가되었습니다.
Wi-Fi 네트워크가 푸시 버튼 설정 이외 방법으로 설정되었습니다.
E-5
무선 라우터의 보안 유형이 다음 중 하나로 설정되었는지 확인합니다. 그렇지 않은 경우 무선 라우터의 보안 유형을 변경한 다음 프린터 네트워크 설정을 재설정합니다.
WEP-64 bit (40 bit)
WEP-128 bit (104 bit)
WPA PSK (TKIP/AES)*
WPA2 PSK (TKIP/AES)*
WPA (TKIP/AES)
WPA2 (TKIP/AES)
WPA3-SAE (AES)
* WPA PSK를 WPA 퍼스널이라고도 합니다. WPA2 PSK를 WPA2 퍼스널이라고도 합니다.
E-6
MAC 주소 필터링이 비활성화되었는지 확인합니다. 활성화된 경우 필터링되지 않도록 프린터의 MAC 주소를 등록합니다. 자세한 내용은 무선 라우터와 함께 제공된 설명서를 참조하십시오. 네트워크 연결 보고서의 Network Status 부분에서 프린터의 MAC 주소를 확인할 수 있습니다.
무선 라우터가 WEP 보안을 이용하는 공유 인증을 사용하고 있을 경우, 인증 키와 인덱스가 올바른지 확인합니다.
무선 라우터의 연결 가능 장치 수가 연결하려는 네트워크 장치 수보다 적을 경우, 무선 라우터를 연결 가능 장치 수가 늘어나도록 설정합니다. 무선 라우터와 함께 제공된 설명서를 참조하여 설정하십시오.
E-8
프린터의 IP 주소 받기 설정이 자동으로 설정된 경우 무선 라우터에서 DHCP를 활성화하십시오.
프린터의 IP 주소 받기 설정이 수동으로 설정된 경우 수동으로 설정한 IP 주소가 범위를 벗어나서(예: 0.0.0.0) 무효화됩니다. 프린터 제어판에서 또는 Web Config를 사용하여 유효한 IP 주소를 설정합니다.
E-9
다음을 확인하십시오.
장치가 켜져 있는지 확인합니다.
인터넷에 액세스하여 프린터에 연결할 장치와 동일한 네트워크 상에 있는 다른 컴퓨터 또는 네트워크 장치에 액세스할 수 있습니다.
위 사항을 확인한 후에도 여전히 프린터와 네트워크 장치를 연결하지 못할 경우, 무선 라우터를 끕니다. 10초 정도 기다렸다가 다시 켭니다. 다음 웹사이트에서 설치 프로그램을 다운로드하고 실행하여 네트워크 설정을 재설정합니다.
http://epson.sn > 설정
E-10
다음을 확인하십시오.
네트워크 상의 다른 장치가 켜져 있는지 여부를 확인합니다.
프린터의 IP 주소 획득을 수동으로 설정한 경우 네트워크 주소(IP 주소, 서브넷 마스크 및 기본 게이트웨이)가 올바른지 여부를 확인합니다.
네트워크 주소가 잘못된 경우 재설정하십시오. 네트워크 연결 보고서의 Network Status 부분에서 IP 주소, 서브넷 마스크 및 기본 게이트웨이를 확인할 수 있습니다.
DHCP가 활성화된 경우 프린터의 IP 주소 받기를 자동으로 변경합니다. IP 주소를 수동으로 설정하려면 네트워크 연결 보고서의 Network Status 부분에서 프린터의 IP 주소를 확인하고 네트워크 설정 화면에서 수동을 선택합니다. 서브넷 마스크를 [255.255.255.0]으로 설정합니다.
여전히 프린터와 네트워크 장치를 연결하지 못할 경우, 무선 라우터를 끕니다. 10초 정도 기다렸다가 다시 켭니다.
E-11
다음을 확인하십시오.
프린터의 TCP/IP 설정 설정을 수동으로 설정한 경우 기본 게이트웨이 주소가 올바른지 여부를 확인합니다.
기본 게이트웨이로 설정된 장치가 켜져 있는지 여부를 확인합니다.
올바른 기본 게이트웨이 주소를 설정합니다. 네트워크 연결 보고서의 Network Status 부분에서 기본 게이트웨이 주소를 확인할 수 있습니다.
E-12
다음을 확인하십시오.
네트워크 상의 다른 장치가 켜져 있는지 여부를 확인합니다.
네트워크 주소(IP 주소, 서브넷 마스크 및 기본 게이트웨이)를 수동으로 입력한 경우 올바른지 여부를 확인합니다.
다른 장치의 네트워크 주소(서브넷 마스크 및 기본 게이트웨이)가 동일한지 여부를 확인합니다.
IP 주소가 다른 장치와 충돌하지 않는지 여부를 확인합니다.
위 사항을 확인한 후에도 여전히 프린터와 네트워크 장치를 연결하지 못할 경우, 다음을 시도합니다.
무선 라우터를 끕니다. 10초 정도 기다렸다가 다시 켭니다.
설치 프로그램을 사용하여 네트워크 설정을 다시 합니다. 다음 웹사이트에서 설치 프로그램을 실행할 수 있습니다.
http://epson.sn > 설정
WEP 보안 유형을 사용하는 무선 라우터에서 여러 개의 암호를 등록할 수 있습니다. 여러 개의 암호를 등록한 경우 첫 번째 등록한 암호가 프린터에서 설정되었는지 확인합니다.
E-13
다음을 확인하십시오.
무선 라우터, 허브 및 라우터와 같은 네트워크 장치가 켜져 있는지 여부를 확인합니다.
네트워크 장치의 TCP/IP 설정이 수동으로 설정되지 않았는지 여부를 확인합니다. (다른 네트워크 장치의 TCP/IP 설정을 수동으로 수행하는 동안 프린터의 TCP/IP 설정이 자동으로 설정된 경우 프린터의 네트워크가 다른 장치의 네트워크와 다를 수 있습니다.)
위의 사항을 확인한 후에도 작동하지 않는 경우 다음을 시도하십시오.
무선 라우터를 끕니다. 10초 정도 기다렸다가 다시 켭니다.
설치 프로그램을 사용하여 프린터와 동일한 네트워크에 있는 네트워크 설정을 수행합니다. 다음 웹사이트에서 설치 프로그램을 실행할 수 있습니다.
http://epson.sn > 설정
WEP 보안 유형을 사용하는 무선 라우터에서 여러 개의 암호를 등록할 수 있습니다. 여러 개의 암호를 등록한 경우 첫 번째 등록한 암호가 프린터에서 설정되었는지 확인합니다.
b: 네트워크 환경의 메시지
메시지
해결 방법
The same SSID may be set on several wireless routers. Check the settings on the wireless router, and change the SSID.
동일한 SSID가 여러 무선 라우터에서 설정될 수 있습니다. 무선 라우터의 설정을 확인하고 SSID를 변경합니다.
The Wi-Fi environment needs to be improved. Turn the wireless router off and then turn it on. If the connection does not improve, see the documentation for the wireless router.
프린터를 무선 라우터에 더 가깝게 이동하고 둘 간의 장애물을 제거한 다음 무선 라우터를 끕니다. 10초 정도 기다렸다가 다시 켭니다. 계속 연결되지 않는 경우 무선 라우터와 함께 제공된 설명서를 참조하십시오.
*No more devices can be connected. Disconnect one of the connected devices if you want to add another one.
동시에 연결할 수 있는 컴퓨터와 스마트 장치는 Wi-Fi Direct(단순 AP) 연결 시 모두 연결됩니다. 다른 컴퓨터 또는 스마트 장치를 추가하려면 먼저 연결된 장치 중 하나의 연결을 끊으십시오.
동시에 연결할 수 있는 무선 장치의 수와 연결된 장치의 수를 네트워크 상태 시트에서 확인할 수 있습니다.
The same SSID as Wi-Fi Direct exists in the environment. Change the Wi-Fi Direct SSID if you cannot connect a smart device to the printer.
프린터 제어판에서 Wi-Fi Direct 설정 화면으로 이동하여 설정을 변경할 메뉴를 선택합니다. 네트워크 이름을 DIRECT-XX-와 같이 변경할 수 있습니다. 32자 이내로 입력합니다.
