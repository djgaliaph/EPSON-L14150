컴퓨터에 SSID를 표시하는 방법

1 . 연결하려는 SSID의 선택 화면을 엽니다.

Windows 10의 경우
작업 표시줄에서 을 클릭합니다.
사용 가능한 SSID가 화면에 표시됩니다.
Windows 8.1/Windows 8/Windows 7의 경우
작업 표시줄에서 을 클릭합니다.
사용 가능한 SSID가 화면에 표시됩니다.
Windows Vista의 경우
작업 표시줄에서 무선 아이콘을 오른쪽 클릭한 다음 사용 가능한 무선 네트워크 보기를 클릭합니다.
사용 가능한 SSID가 무선 네트워크에 연결 화면에 표시됩니다.
Windows XP의 경우
시작 > 제어판 > 네트워크 연결 > 새 연결 만들기를 클릭합니다.
사용 가능한 SSID가 무선 네트워크 연결 화면에 표시됩니다.
MacOS의 경우
메뉴를 클릭합니다.
사용 가능한 SSID가 화면에 표시됩니다.

2 . 프린터의 제어판에 표시된 대로 동일한 SSID를 선택합니다.
