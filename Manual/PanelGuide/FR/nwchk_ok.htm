Le diagnostic de connexion réseau est un Succès, mais l'imprimante ne peut pas se connecter

L'imprimante et le routeur sans fil sont connectés avec succès, mais l'imprimante ne peut pas être connectée à l'ordinateur via un routeur sans fil. Essayez les solutions suivantes, dans l'ordre.
Contrôlez la fonction de séparateur de confidentialité sur le routeur sans fil
Il s'agit d'un problème courant sur les routeurs sans fil mobiles. Une fonction de séparateur de confidentialité coupant la communication entre les appareils est activée sur le routeur sans fil. Il s'agit d'une fonction de sécurité pour couper la communication entre les appareils via le routeur. Lorsqu'elle est activée, la connexion est impossible, car la communication entre l'ordinateur et l'imprimante est coupée. Désactivez la fonction de séparateur de confidentialité sur le routeur sans fil, puis définissez à nouveau le réseau. Consultez le manuel de votre routeur sans fil pour plus de détails sur la méthode de configuration. En fonction du fabricant, la fonction de séparateur de confidentialité peut avoir un nom différent.
