Comment afficher un SSID sur un ordinateur

1 . Ouvrez l'écran de sélection du SSID auquel vous souhaitez vous connecter.

Pour Windows 10
Cliquez sur dans la barre des tâches.
Les SSID disponibles s'affichent à l'écran.
Pour Windows 8.1/Windows 8/Windows 7
Cliquez sur dans la barre des tâches.
Les SSID disponibles s'affichent à l'écran.
Pour Windows Vista
Faites un clic droit sur l'icône sans fil dans la barre des tâches, puis cliquez sur Afficher les réseaux sans fil disponibles.
Les SSID disponibles s'affichent à l'écran Connexion au réseau sans fil.
Sous Windows XP
Cliquez sur Démarrer > Panneau de configuration > Connexions réseau > Créer une nouvelle connexion.
Les SSID disponibles s'affichent à l'écran Connexion à un réseau sans fil.
Sur MacOS
Cliquez sur le menu .
Les SSID disponibles s'affichent à l'écran.

2 . Sélectionnez le même SSID que celui affiché au panneau de configuration de l'imprimante.
