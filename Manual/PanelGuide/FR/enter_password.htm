Saisie du mot de passe et connexion

1 . Connectez-vous au SSID.

Pour Windows 10
Décochez Connexion automatique, puis cliquez sur Connexion.
Pour Windows 8.1/Windows 8/Windows 7/Windows Vista
Cliquez sur Connexion.
Sous Windows XP
Faites un clic droit sur le SSID, puis cliquez sur Connexion.
Sur MacOS
Aucune action nécessaire.

2 . Connectez l'imprimante à l'ordinateur en saisissant dans l’ordinateur le mot de passe affiché sur le panneau de configuration de l'imprimante.

Pour Windows XP, saisissez la Clé réseau et la Confirmation de clé réseau.

3 . Pour les utilisateurs de Windows 8.1/Windows 8/Windows 7/Windows Vista, vous pouvez configurer votre ordinateur pour qu'il se connecte à l'imprimante automatiquement à chaque fois que vous démarrez votre ordinateur.

Vous n'avez pas besoin de régler pas cette option si vous souhaitez vous connecter à l'imprimante uniquement lorsque vous l'utilisez.
