Diagnosis sambungan jaringan adalah Lulus, namun printer tidak dapat tersambung

Printer dan router nirkabel berhasil tersambung, namun printer tidak dapat disambungkan ke komputer melalui router nirkabel. Coba solusi berikut secara berurutan.
Periksa fungsi pemisah privasi pada router nirkabel
Ini merupakan masalah umum untuk router nirkabel perangkat bergerak. Fungsi pemisah privasi yang memutuskan komunikasi di antara perangkat diaktifkan di router nirkabel. Ini adalah fitur keamanan untuk memutuskan komunikasi di antara perangkat melalui router. Jika diaktifkan, printer dan komputer tidak dapat disambungkan karena komunikasi terputus. Nonaktifkan fungsi pemisah privasi pada router nirkabel, lalu atur kembali jaringan. Untuk rincian tentang metode pengaturan, lihat panduan router nirkabel Anda. Bergantung pada produsen, fungsi pemisah privasi mungkin memiliki nama yang berbeda.
