Memasukkan Sandi dan Membuat Sambungan

1 . Sambungkan ke SSID.

Untuk Windows 10
Hapus Sambung otomatis, lalu klik Sambung.
Untuk Windows 8.1/Windows 8/Windows 7/Windows Vista
Klik Sambung.
Untuk Windows XP
Klik kanan SSID, lalu klik Sambung.
Untuk MacOS
Tidak diperlukan tindakan.

2 . Sambungkan printer dan komputer dengan memasukkan sandi yang ditampilkan pada panel kontrol printer di komputer.

Untuk Windows XP, masukkan Kunci jaringan, lalu Konfirmasikan kunci jaringan.

3 . Untuk pengguna Windows 8.1/Windows 8/Windows 7/Windows Vista, Anda dapat mengatur komputer agar tersambung ke printer secara otomatis setiap kali menghidupkan komputer.

Anda tidak perlu mengaturnya jika hanya ingin menyambung ke printer saat menggunakannya.
