パスワードの入力と接続

1 . SSIDに接続を開始します。

Windows 10の場合
［自動的に接続］のチェックを外し、［接続］をクリックします。
Windows 8.1/Windows 8/Windows 7/Windows Vistaの場合
［接続］をクリックします。
Windows XPの場合
SSID上で右クリックし、［接続］を選択します。
MacOSの場合
操作はありません。

2 . プリンターの操作パネルに表示されているパスワードをコンピューターに入力して接続します。

Windows XPでは、［ネットワークキー］と［ネットワークキーの確認］それぞれに入力してください。

3 . Windows 8.1/Windows 8/Windows 7/Windows Vistaは、コンピューターの起動時に自動でプリンターと接続するかを設定できます。

印刷するときだけプリンターを接続する場合は、この設定は不要です。
