Τρόπος εμφάνισης ενός SSID στον υπολογιστή

1 . Ανοίξτε την οθόνη επιλογής του SSID, στο οποίο θέλετε να συνδεθείτε.

Για τα Windows 10
Κάντε κλικ σε στη γραμμή εργασιών.
Τα διαθέσιμα SSID εμφανίζονται στην οθόνη.
Για Windows 8.1/Windows 8/Windows 7
Κάντε κλικ σε στη γραμμή εργασιών.
Τα διαθέσιμα SSID εμφανίζονται στην οθόνη.
Για Windows Vista
Κάντε δεξί κλικ στο εικονίδιο του ασύρματου δικτύου στη γραμμή εργασιών και, στη συνέχεια, στην επιλογή Προβολή διαθέσιμων ασύρματων δικτύων.
Τα διαθέσιμα SSID εμφανίζονται στην οθόνη Σύνδεση στο ασύρματο δίκτυο.
Για Windows XP
Κάντε κλικ σε Έναρξη > Πίνακας ελέγχου > Συνδέσεις δικτύου > Δημιουργία νέας σύνδεσης.
Τα διαθέσιμα SSID εμφανίζονται στην οθόνη Σύνδεση ασύρματου δικτύου.
Για MacOS
Κάντε κλικ στο μενού .
Τα διαθέσιμα SSID εμφανίζονται στην οθόνη.

2 . Επιλέξτε το ίδιο SSID με αυτό που εμφανίζεται στον πίνακα ελέγχου του εκτυπωτή.
