Εισαγωγή κωδικού και σύνδεση

1 . Σύνδεση στο SSID.

Για τα Windows 10
Αποεπιλέξτε το πλαίσιο Σύνδεση αυτόματα και στη συνέχεια κάντε κλικ στην επιλογή Σύνδεση.
για Windows 8.1/Windows 8/Windows 7/Windows Vista
Κάντε κλικ στην επιλογή Σύνδεση.
Για Windows XP
Κάντε δεξί κλικ στο SSID και στη συνέχεια στην επιλογή Σύνδεση.
Για MacOS
Δεν απαιτείται καμία ενέργεια.

2 . Συνδέστε τον εκτυπωτή και τον υπολογιστή εισάγοντας τον κωδικό που εμφανίζεται στον πίνακα ελέγχου του εκτυπωτή στον υπολογιστή.

Για Windows XP, πραγματοποιήστε την εισαγωγή στο πεδίο Κλειδί δικτύου και στο Επιβεβαίωση κλειδιού δικτύου.

3 . Για χρήστες Windows 8.1/Windows 8/Windows 7/Windows Vista, μπορείτε να ρυθμίσετε τον υπολογιστή σας να συνδέεται με τον εκτυπωτή αυτόματα κάθε φορά που ενεργοποιείτε τον υπολογιστή σας.

Μην προβείτε στη ρύθμιση αυτή, αν θέλετε η σύνδεση με τον εκτυπωτή να γίνεται μόνο όταν τον χρησιμοποιείτε.
