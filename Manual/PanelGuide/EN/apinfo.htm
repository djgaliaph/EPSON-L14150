Checking the SSID and the Password for the Wireless Router

To find the SSID and password, check the back or bottom of the wireless router that you want to connect to, or the documentation provided with it, such as an information card, label, or manual.
If an SSID is displayed with a name that is similar to the one that you want to connect to, for example if the SSID name differs by a letter such as a or g, or a number such as 2.4 or 5, select it.
If the SSID that you want to connect to is not displayed, follow the instructions on the printer's control panel to update the SSIDs list by searching again. If it is still not displayed, enter the SSID manually.
