การใส่รหัสผ่านและการเชื่อมต่อ

1 . เชื่อมต่อกับ SSID

สำหรับ Windows 10
ล้างข้อมูล เชื่อมต่อโดยอัตโนมัติ จากนั้นคลิกที่ เชื่อมต่อ
สำหรับ Windows 8.1/Windows 8/Windows 7/Windows Vista
คลิกที่ เชื่อมต่อ
สำหรับ Windows XP
คลิกขวาที่ SSID จากนั้นคลิกที่ เชื่อมต่อ
สำหรับ MacOS
ไม่ต้องดำเนินการใด ๆ

2 . เชื่อมต่อเครื่องพิมพ์เข้ากับคอมพิวเตอร์โดยการป้อนรหัสผ่านที่แสดงอยู่บนแผงควบคุมของเครื่องพิมพ์เข้าไปที่เครื่องคอมพิวเตอร์

สำหรับ Windows XP ให้ป้อน รหัสเครือข่าย และ ยืนยันรหัสเครือข่าย

3 . สำหรับผู้ที่ใช้งาน Windows 8.1/Windows 8/Windows 7/Windows Vista คุณสามารถตั้งค่าการเชื่อมต่อคอมพิวเตอร์ของคุณเข้ากับเครื่องพิมพ์ได้โดยอัตโนมัติทุกครั้งที่คุณเปิดใช้งานเครื่องคอมพิวเตอร์ของคุณ

คุณต้องไม่ตั้งค่านี้หากคุณต้องการเชื่อมต่อเฉพาะเวลาที่คุณต้องการใช้งานเครื่องพิมพ์เท่านั้น
