Parolun Daxil Edilməsi və Qoşulma

1 . SSID-yə qoşulun.

Windows 10 üçün
Avtomatik qoşulu təmizləyin, sonra isə Qoşul düyməsini basın.
Windows 8.1/Windows 8/Windows 7/Windows Vista üçün
Qoşul klikləyin.
Windows XP üçün
SSID-ni sağla klikləyin, sonra isə Qoşul düyməsini basın.
MacOS üçün
Əməliyyat tələb edilmir.

2 . Printerin idarəetmə panelindəki parolu kompüterə daxil etməklə printeri və kompüteri qoşun.

Windows XP üçün Şəbəkə açarı və Şəbəkəni təsdiqlə açarını daxil edin.

3 . Windows 8.1/Windows 8/Windows 7/Windows Vista istifadəçiləri üçün, kompüterinizi hər dəfə kompüteri yandırdığınız zaman printerə avtomatik qoşulmağa qura bilərsiniz.

Printerə yalnız ondan istifadə etdiyiniz zaman qoşulmaq istəyirsinizsə, bunu qurmalı deyilsiniz.
